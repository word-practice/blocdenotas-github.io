TAGS

Las tags de HTML también sirven para transmitir información a los distintos navegadores que sirve para que tu web esté mejor estructurada y pueda visualizarse en todos los navegadores.

Las etiquetas HTML están formadas por una etiqueta de inicio y una de cierre. Los comandos HTML van dentro de antilambdas (“<” y “>”) y contienen la información. La etiqueta final va tachada con una barra. El conjunto compuesto por etiqueta de apertura, información y etiqueta de cierre se llama elemento. Puedes asignar un atributo al elemento, que aportará información adicional.

¿QUE DEFINE LA ESTRUCTURA DE UNA PAGINA WEB?

Lo que definen las estructura son los tags, los tags en HTML organizan y estructuran el contenido de una página web, incluyendo secciones, encabezados, párrafos y otros elementos. La estructura básica para la página web incluye tags como HTML, HEAD, BODY, y DIV, que ayudan a organizar el contenido de manera jerárquica y lógica.

Elementos comunes: h1..h6, p, pre, br, hr, a (href), , img (src, width, height),style (background-color, color, font, size), sub, sup, small, i, table, list.

  • h1 a h6: Representan encabezados de diferentes niveles de importancia, donde (h1) es el más importante y los que van descendiendo hasta (h6) son menos importante.

  • p: Define un párrafo de texto.

  • pre: Muestra texto preformateado, respetando espacios y saltos de línea.

  • br: Inserta un salto de línea, permitiendo separar texto sin crear un nuevo párrafo.

  • hr: Inserta una línea horizontal para dividir contenido.

  • href: Define un hipervínculo, que permite al usuario navegar a otra página o recurso al hacer clic.

  • img (src, width, height): Inserta una imagen en la página web. El atributo src especifica la fuente de la imagen, mientras que width y height controlan sus dimensiones.

  • style (background-color, color, font-size): Aplica estilos al contenido, permitiendo personalizar la apariencia de elementos, como el color de fondo, el color del texto, y el tamaño de la fuente.

  • sub: Muestra texto en subíndice.

  • sup: Muestra texto en superíndice.

  • small: Muestra texto en un tamaño más pequeño que el texto circundante.

  • i: Muestra texto en cursiva.

  • table: Define una tabla, usada para mostrar datos en filas y columnas.

  • list: Define listas de ítems, que pueden ser ordenadas (ol) o no ordenadas (ul).
